

## Getting Started

To get a local copy up and running, please follow these simple steps.

### Prerequisites

Here is what you need to be able to run the project.

- Node.js (Version: >=18.x)
- Pnpm _(recommended)_


## Development

1. Install packages

   ```sh
   pnpm install
   ```

2. Run the project

   ```sh
   pnpm dev

   # Running on http://localhost:3000/
   ```
